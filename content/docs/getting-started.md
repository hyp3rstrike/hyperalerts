# Getting Started

Hyper Alerts is a local-first alert overlay app for OBS and Streamer.bot.

## Install

1. Download the latest release from the Download page.
2. Run `HyperAlerts.exe`.
3. In OBS, add a Browser Source pointing to `http://127.0.0.1:3000/`.

## Streamer.bot Patch

1. Open the Hyper Alerts **Tools** panel.
2. Select your Streamer.bot install.
3. Click **Apply Hyper Alerts Patch**.

## Notes

- Keep the app open while streaming.
- Update checks run on startup.
- Your media and config stay local.
