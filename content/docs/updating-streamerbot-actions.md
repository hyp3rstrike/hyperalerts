# Updating Streamer.bot Actions

Periodically, a release will ship with an updated **Streamer.bot Action**. Updates can be pushed to your Streamer.bot from **Hyper Alerts** with one click using the instructions below.

## What to do

1. Open `HyperAlerts.exe`.
2. Click the **settings cog-wheel** in the **bottom left (on the sidebar)**.
3. Click **Browse actions.json** and locate the file under `data\actions.json` in your Streamer.bot directory.
4. Click **Apply Hyper Alerts Patch**.
5. Re-open Streamer.bot.

## Notes

- Streamer.bot should be closed before patching.
- You can do it with Streamer.bot open, but you'll need to restart for the new changes to show.
- The Hyper Alerts action is versioned with `v{major}.{minor}.{revision}` format.
